# Content Not Loading

Troubleshooting guide for issues accessing biblical resources and content.

<Callout type="info" title="Beta Documentation">
Content loading issues and solutions will be documented during beta testing.
</Callout>

## Common Content Issues

### Resources Won't Load

**Symptoms:**
- Blank pages when browsing
- "Loading..." never completes
- Error messages when accessing content

**Solutions:**

1. **Check Connection Quality**
   - Ensure strong WiFi signal (3+ bars)
   - Move closer to RKPi5 device
   - Disconnect and reconnect to WiFi

2. **Clear Browser Cache**
   ```
   Chrome: Settings → Privacy → Clear browsing data
   Safari: Settings → Clear History and Website Data
   Firefox: Options → Privacy → Clear Data
   ```

3. **Try Different Browser**
   - Test with another browser
   - Use incognito/private mode
   - Update browser to latest version

### Video/Audio Won't Play

**Symptoms:**
- Video player shows error
- Audio files won't start
- Media loads but won't play

**Solutions:**

1. **Check Media Format Support**
   <Callout type="info">
   All content is provided in widely supported formats.
   </Callout>
   
   - Video: MP4 (H.264)
   - Audio: MP3
   - Supported by all modern browsers

2. **Browser Permissions**
   - Allow autoplay in browser settings
   - Enable media playback
   - Check volume isn't muted

3. **Bandwidth Issues**
   - Too many users streaming simultaneously
   - Try audio-only instead of video
   - Download content for offline viewing (if available)

### Search Not Working

**Symptoms:**
- Search returns no results
- Search is very slow
- Search crashes or times out

**Solutions:**

1. **Verify Search Terms**
   - Check spelling
   - Try broader search terms
   - Use quotes for exact phrases: "love one another"

2. **Rebuild Search Index**
   - Access admin panel
   - Go to Content Management
   - Click "Rebuild Search Index"
   - Wait for process to complete (5-10 minutes)

3. **Browser Compatibility**
   - Ensure JavaScript is enabled
   - Disable browser extensions temporarily
   - Try different browser

## Content Update Issues

### Updates Won't Download

**Symptoms:**
- Update fails to start
- Update gets stuck partway through
- Error during update process

**Solutions:**

1. **Check Internet Connection**
   <Callout type="warning">
   Content updates require internet connectivity.
   </Callout>
   
   - Connect RKPi5 to WiFi network with internet
   - Verify internet is working (try browsing)
   - Check for sufficient bandwidth

2. **Check Storage Space**
   ```
   Admin Panel → System → Storage
   Required: 10GB free space for updates
   Current: Check available space
   ```

3. **Retry Update**
   - Cancel current update
   - Restart RKPi5 device
   - Initiate update again

### Content Appears Corrupted

**Symptoms:**
- Garbled text or strange characters
- Videos show artifacts or glitches
- Audio has static or skips

**Solutions:**

1. **Verify Content Integrity**
   - Admin Panel → Content Management → Verify Files
   - This will check for corrupted files
   - May take 10-15 minutes

2. **Re-download Affected Content**
   - Note which resources are problematic
   - Use "Re-download" option in admin panel
   - Targets specific content without full reset

3. **Factory Reset (Last Resort)**
   <Callout type="error" title="Warning">
   Factory reset will restore original content but erase custom settings.
   </Callout>
   
   - Backup any custom configurations
   - Admin Panel → System → Factory Reset
   - Follow on-screen instructions

## Performance Issues

### Slow Loading Times

**Expected Performance:**
- Page loads: Under 2 seconds
- Video start: 3-5 seconds
- Search results: Under 1 second

**Solutions:**

1. **Check Active Users**
   - Admin Panel → Users → Active Connections
   - Performance degrades with 10+ users
   - Ask some users to disconnect temporarily

2. **Restart Services**
   - Admin Panel → System → Restart Services
   - Clears cache and resets connections
   - No need to power cycle device

3. **Check System Resources**
   ```
   Admin Panel → System → Performance
   CPU Usage: Should be under 70%
   Memory: Should have 1GB+ free
   Temperature: Should be under 70°C
   ```

### Content Database Issues

**Symptoms:**
- Missing content that should be there
- Duplicate entries
- Broken links or references

**Solutions:**

1. **Rebuild Content Database**
   - Admin Panel → Content Management
   - Click "Rebuild Database"
   - This re-indexes all content
   - Takes 15-20 minutes

2. **Check for Incomplete Updates**
   - Review update history
   - Complete any pending updates
   - Verify update success

## Getting Help

If content issues persist:

1. **Document the Problem**
   - Which content is affected?
   - When did the issue start?
   - What have you tried?
   - Screenshot any error messages

2. **Check System Logs**
   - Admin Panel → System → Logs
   - Look for errors related to content
   - Note timestamps and error codes

3. **Contact Support**
   - Visit [Contact Page](/contact)
   - Provide documentation above
   - Include device serial number
