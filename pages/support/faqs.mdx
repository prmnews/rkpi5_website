## About RKPi5

### What is RKPi5?

RKPi5 is a portable, self-contained biblical resource platform based on Raspberry Pi 5. It provides 26GB+ of biblical content including audio, video, and text resources that work completely offline.

### Who is RKPi5 for?

RKPi5 is designed for:

- Home Bible study groups
- Missionaries in remote locations
- Churches preparing for potential disruptions
- Families building emergency preparedness
- Anyone wanting offline access to biblical resources

### Do I need internet to use RKPi5?

<Callout type="success" title="Completely Offline">
No internet connection required for normal operation.
</Callout>

RKPi5 works completely offline. Internet is only needed for:
- Initial setup (optional)
- Content updates (optional)
- Downloading DIY scripts (Hobbyist tier only)

## Getting Started

### How long does setup take?

Setup takes less than 5 minutes:

1. Power on device (30 seconds)
2. Connect to WiFi network (1 minute)
3. Access portal (instant)
4. Start using resources (immediately)

### What devices can connect?

Any WiFi-enabled device:
- Smartphones (iPhone, Android)
- Tablets (iPad, Android, Kindle)
- Laptops (Windows, Mac, Linux)
- Desktop computers
- Chromebooks

### How many people can use it at once?

RKPi5 supports up to **15 concurrent users** connecting to the same device.

## Technical Questions

### How long does the battery last?

**6+ hours** of typical use.

Factors affecting battery life:
- Number of connected users (more users = faster drain)
- Content type (video uses more than audio)
- WiFi signal strength

### What's the WiFi range?

Approximately **100 feet (30 meters)** in open space.

Actual range depends on:
- Physical obstacles (walls, furniture)
- Interference from other devices
- Building materials

### Can I use it while charging?

Yes! RKPi5 works normally while charging. There's no need to power down for charging.

### What happens if battery dies while in use?

RKPi5 is designed to shut down safely when battery is low:

1. Warning at 20% battery
2. Warning at 10% battery
3. Safe shutdown at 5%

No data loss or corruption from normal shutdown.

## Security & Privacy

### Is my data private?

<Callout type="success" title="Privacy First">
RKPi5 operates completely offline. No data leaves your device.
</Callout>

- No cloud connection required
- No usage tracking
- No data collection
- No accounts or registration needed (except for purchases)

### Is the WiFi network secure?

Yes, the WiFi network is password-protected using WPA2 encryption.

For sensitive environments:
- Change default password
- Disable network when not in use
- Use MAC address filtering (advanced users)

### Can it be used in countries with restrictions?

RKPi5 operates entirely offline with no external communications, making it ideal for restrictive environments. However, check local laws regarding:

- WiFi device operation
- Content restrictions
- Encryption regulations
