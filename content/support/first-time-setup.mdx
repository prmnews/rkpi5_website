# First Time Setup

Complete setup guide for configuring your RKPi5 for the first time.

<Callout type="info" title="Beta Documentation">
Detailed step-by-step instructions with screenshots will be available after beta testing.
</Callout>

## Prerequisites

Before you begin:

- [ ] RKPi5 device fully charged
- [ ] Computer or mobile device with WiFi
- [ ] Quick start card with credentials

## Setup Process

### Step 1: Initial Power On

Power on your RKPi5 and wait for the system to boot completely.

<CodeBlock title="Expected Boot Time" language="text">
Average boot time: 30-45 seconds
LED Status: Flashing green = booting
LED Status: Solid green = ready
</CodeBlock>

### Step 2: Network Connection

1. Open WiFi settings on your device
2. Look for network named "RKPi5-XXXX"
3. Enter the password from your quick start card
4. Wait for connection confirmation

### Step 3: Access Admin Portal

Open your web browser and navigate to:

```
http://rkpi5.local/admin
```

### Step 4: Initial Configuration

<Callout type="warning">
You'll be prompted to change the default password on first login.
</Callout>

Configure the following settings:

- **WiFi Network Name**: Customize your network SSID
- **Network Password**: Set a strong password
- **Admin Password**: Change from default
- **Time Zone**: Set your local time zone
- **Content Updates**: Configure auto-update preferences

## Verification

After setup, verify everything is working:

1. Access the main portal at `http://rkpi5.local`
2. Browse available resources
3. Test video and audio playback
4. Check battery status in admin panel

## Troubleshooting

If you encounter issues:

- **Can't find WiFi network**: Ensure device is powered on and LED is solid green
- **Can't access portal**: Try `192.168.4.1` instead of `rkpi5.local`
- **Forgot password**: Contact support for password reset instructions
